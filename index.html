About Is FFBID

Fest From Block kali ini di adakan dengan Tour Server yang akan di adakan di server Arknesia, festival minecraft online pertama di indonesia dengan mengundang banyak DJ yang pastinya keren!.

Fest From Block dapat dinikmati dengan cara masuk discord kita! makanya jangan lupa untuk masuk Discord Arknesia Server untuk dapat bertemu dan mendengarkan musik DJ-Lineup kita.

How to join events

Halo kamu yang belum tau cara join events ini, kamu dapat menggunakan Bedrock Edition/Pocket Edition dan juga Java Edition Versi 1.16+ loh, caranya tinggal tambahkan alamat ip server play.arknesia.net, kamu bisa mencontohnya di gambar di atas ya!, setelah itu kamu dapat mendengarkan musik ini disini maupun via discord server ya

Khusus untuk penggua Java Edition nih, kamu bisa mengenable server resourcepack ya agar dapat menikmati events dengan vibe yang lebih asik!
